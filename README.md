# 🔍 Buscador de Imagens

Uma aplicação web simples e elegante para buscar imagens em múltiplos mecanismos de busca simultaneamente.

## ✨ Funcionalidades

- **Múltiplos mecanismos de busca**: Pesquise imagens em 9 plataformas diferentes
  - Google Imagens
  - Bing Imagens
  - Yahoo Imagens
  - Yandex
  - Pixabay
  - Pexels
  - Unsplash
  - Freepik
  - Pinterest
- **Interface elegante**: Design sofisticado com estética vintage
- **Responsivo**: Funciona perfeitamente em dispositivos móveis e desktop
- **Abertura em nova aba**: Resultados abrem em uma nova aba para melhor experiência

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Digite o termo de busca no campo de pesquisa
3. Selecione o mecanismo de busca desejado
4. Clique em **"Buscar"** para abrir os resultados
